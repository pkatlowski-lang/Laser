Publikuj na GitHub

Wyślij zmiany do repo — widoczne dla wszystkich
🚀 Co się stanie po kliknięciu "Publikuj":
  1. Panel połączy się z GitHub API i zaktualizuje plik data.json w Twoim repo
  2. Po 30-60 sekundach GitHub Pages odbuduje stronę z nowymi treściami
  3. Wszyscy odwiedzający zobaczą zmiany przy następnym załadowaniu strony

Jeśli nie skonfigurowałeś połączenia, przejdź najpierw do Ustawień.

🔗 Połączenie z GitHub

Wypełnij dane swojego repozytorium aby panel mógł publikować zmiany
Jak wygenerować Personal Access Token (PAT):
  1. Wejdź na github.com/settings/personal-access-tokens/new
  2. Wybierz Fine-grained tokens (zalecane)
  3. Token name: LaserArt Admin, expiration: wedle uznania
  4. Repository access: Only select repositories → wybierz swoje repo
  5. Permissions: rozwiń Repository permissions → Contents → ustaw na Read and write
  6. Kliknij Generate token i skopiuj wygenerowany ciąg (zaczyna się od github_pat_...)
